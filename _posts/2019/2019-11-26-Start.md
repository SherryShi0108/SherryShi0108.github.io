## 写在开头

​一直想创建一个自己的博客，想着有时间看看html css js等相关技术自己写个博客。总觉得要花很多时间但是一直没抽出时间，搭建博客也就一直处于未完成状态。某天豁然醒悟，花了一天时间直接参考模板搭建了这个托管在Github上的博客，在此特别感谢模板[作者]( https://github.com/Gaohaoyang/gaohaoyang.github.io )。也警醒自己凡事不能等到准备工作全部准备好了后去干，先干了再说！博客可能之后看情况会进行更改，但是目前最重要的是要开始写博客啦~





## 内容

博客内容计划分成以下几个部分：

+ 【001】计算机基础
+ 【002】数学基础
+ 【003】编程语言
+ 【004】技术综合
+ 【005】翻译合集
+ 【006】扩展阅读
+ 【007】个人记录
+ 【008】其他

​基础很重要，要努力夯实基础，计算机基础【001】主要包含五大内容：数据结构与算法、计算机组成原理、操作系统、计算机网络、编译原理。据说编程到最后都是在解决数学问题，所以数学方面的知识也不能拉下，希望能把数学知识好好了解一下，数学基础【002】存放数学相关知识。编程语言【003】是工具，要善于利用各种工具，目前计划的是熟悉C#、C、Python、Java这些对我来说正在使用或者曾经学习过的语言。技术综合【004】中计划是放入一些技术细节和其他无法归类到第一第三类的技术内容。由于目前很多技术知识的源头来源于西方，所以英文一定要学好，翻译合集【005】计划对一些自己看过的一些英文文档进行翻译，也算为技术的传播贡献绵薄之力，也会对一些实时新闻、经典问题与解答进行翻译训练自己的翻译能力。扩展阅读【006】计划将自己阅读的一些非专业书籍、论文等进行记录。个人纪录【007】^回是个人相关的一些内容，比如思想、观点、计划、随笔等。把一些暂时无法分类的内容放入其他【008】这一类中，之后构建更完善的分类表时再进行准确地划分。



## 写作准则

第一次写博客对于此还比较陌生，先定下一些写作准则，之后写博客尽量按照这些准则进行。之后又需要增删改的地方逐渐优化~

+ 踏实真诚写作，尽量能讲清楚想表达的内容
+ 每篇博客写好引用参考和更新历史
